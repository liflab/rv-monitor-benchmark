This lab compares various runtime monitors on event traces.

Monitors included

  • BeepBeep 1.7.8, a monitor for LTL-FO+ formulas on XML events.
  • BeepBeep 3 (v0.10.3), a general-purpose event stream processing library and the successor to BeepBeep 1.

Scenarios

The lab is composed of a number of scenarios, which are basic experimental units made of:

  • A trace of events representing a particular use case
  • A property to evaluate on this trace of events

For a given scenario and a given tool, the lab provides the specification corresponding to the property to evaluate in the input notation/language of the tool, and optionally a conversion function to turn events of the input trace into data structures in the format accepted by the tool in question.

Measurements

For each scenario and each monitor, the lab performs the following measurements:

  • The maximum memory consumption used by the monitor
  • The total evaluation time

It also measures the memory consumption and running time at periodical intervals during the evaluation. From these measurements, the lab produces:

  • A cumulative table of the running time/memory consumption of each monitor on each scenario
  • Line plots of running time/memory for each scenario
